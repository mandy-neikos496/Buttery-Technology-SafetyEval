# SafetyEval Archive

This directory preserves historical development files that are not part of the current installable package or final evaluation workflow.

- `backups/`: backups created before repairing run logs
- `cli_reports/`: HTML reports produced during CLI testing
- `development/`: prototype and NVIDIA smoke-test code
- `docs/`: historical progress documentation
- `runs/`: intermediate multi-model evaluation logs
- `scripts/`: earlier development and sample-evaluation scripts

The final consolidated evaluation data remains in `safetyeval_pkg/realeval/run_final_consolidated.jsonl`.
